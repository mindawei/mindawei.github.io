    优胜奖

    核心成员
    阿里中间件性能挑战赛
    2017.05 - 2017.07

    初赛要求参赛者实现进程内消息引擎，主要优化如下：

    • 1) 数据压缩进行 IO 优化
    • 2) 按线程粒度存文件，进行无锁化处理

    复赛要求参赛者模拟数据同步，主要优化如下：

    • 1) 并行化处理（流水线）
    • 2) IO 优化（读取文件大小调参、网路传输自定义格式）
    • 3) 预测技术（局部相似性原理）
    • 4) map 优化（分桶减少 hash 冲突）
    • 5) 减少 GC 和数据拷贝

    本人在比赛中复责主体代码实现。最终进入总决赛，获得优胜奖

    总决赛第 14 名

    队长
    华为软件精英挑战赛
    2017.03 - 2017.05

    比赛要求很短时间内给出一种视频服务器的部署方案，满足消费者需求并使得成本最小。

    本人在比赛中负责主体 Java 代码编写、任务分工。

    尝试过遗传算法、模拟退火等启发式方法，效果不佳，最后为了提高计算速度，采取这些方法：

    • 1) 不使用 JDK 容器,自己实现相关容器
    • 2) 缓存部分预计算，先求粗糙解，再求精确解
    • 3) 引入 BP 神经网络训练节点参数
    • 4) Djkstra 最短路中使用堆优化
    • 5) 使用 zkw 替换普通的最小费用流

    最终获得杭厦赛区一等奖，进入总决赛并获得第 14 名（16 强中唯一使用 Java 开发的队伍）。
